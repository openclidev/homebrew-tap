# Install `ocli`

The OpenCLI CLI to help you manage OpenCLI spec-compliant documents.

Checkout the [official OpenCLI repository](https://github.com/bcdxn/opencli) and the official [OpenCLI Specification docs](https://opencli.dev) for more details.

```
brew tap openclidev/tap
```

```
brew trust openclidev/tap
```

```
brew install ocli
```
